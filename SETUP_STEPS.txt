STEP 3: SETUP BACKEND (Local)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. Open "backend" folder
  2. Click address bar → type "cmd" → Enter
  3. Run: npm install
  4. Run: copy .env.example .env
  5. Run: notepad .env
     → Change DB_PASS to your Railway password
     → Save and close
  6. Run: npm run seed
     → Should say "Seed complete!" ✅
  7. Run: npm start
     → Should say "MedEase API running on port 5000" ✅
